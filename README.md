# Keeper App

A simple note-taking web app inspired by Google Keep, built using React.
Users can add notes and delete them dynamically — showcasing React’s component-based UI and state management.

# Features
- Add new notes
- Delete existing notes
- Built using React components

# Built With
- React
- JavaScript (ES6)
- CSS
- HTML
